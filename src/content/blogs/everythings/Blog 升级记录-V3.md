## 背景

之前使用 kuizuo 大佬魔改的 Docusaurus 框架的 blog 主题改了 V2 版本的 blog 网站，用了差不多一年时间。总体感觉非常不错，但是对于我来说，有下面几个问题：

1. 编译速度慢；
2. 总体太重了；
3. 博客内容不够简洁干净，我更喜欢简洁清爽些的页面。

基于以上几点，加上现在 AI 大行其道，Vibe Coding 深入人心，于是我便尝试使用 AI 重构 Blog V3 版本。

## AI 工具

1. VSCode 的 Github Copilot 插件；
2. Github Copilot Claude Sonnet 4.5；
3. Vscode。

万事俱备，只欠东风。开整！！！

## 过程

...... 此处省略 n 万字。

所有的模块和代码，除了文章之外都由 AI 生成，从 webpack 换成了 Vite。启动和编译速度快了 N 倍。主题用纯黑色和纯白色，页面简洁清爽。

因为是从头 1 对 1 整的，所以如意如意，按我心意。一切都比较符合我的想法。

## 结果

整体效果，深得朕心。加入了百度页面浏览分析和 Algolia Search，使用 Github Pages 部署。
