## 🎉 牧生 Blogs

Build with React + Doucsaurus.

## 📥 运行

```bash
git clone https://github.com/yuluo-yx/blog.git
cd blog
make preview
```

> 如果 sharp 安装失败
> 需要设置 npm 和 pnpm 的源，并且设置 npm 的 https-proxy 和 proxy 

Notes: 可能需要在 vscode 中设置 eslint 插件。

## 🛠️ 构建

```bash
pnpm build
```
